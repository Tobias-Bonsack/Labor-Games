# Labor-Games WS21/22

## Chemie-Engine
Inspieriert von Zelda: A Breath of the wild

#### Basics vom Spiel

1. Chemie Eigenschaften
   - Hitze (Feuer)
   - Kälte (Eis)
   - Elektrizität (Blitze)
2. Ansicht
   - Third-Person Ansicht (Cinemachine)
3. Spiel
   - Aufbau wie eine Prüfung

#### Zusätzliches

1. Chemie Eigenschaften
   - Feuchtigkeit (Pflanzenwachstum, Holzwaffen nicht brennbar)
   - Säure (Korrosion)
